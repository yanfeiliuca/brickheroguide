LEGO Batman Legacy of the Dark Knight — PC System Requirements

Wondering if your PC can run LEGO Batman: Legacy of the Dark Knight? LEGO games are typically well-optimized and accessible on a wide range of hardware, and this entry is no exception. Here are the official system requirements, plus real-world performance tips.

Minimum System Requirements (1080p / 30 FPS / Low Settings)

OSWindows 10 64-bit (version 21H2 or newer)
CPUIntel Core i5-4590 / AMD Ryzen 3 1200
RAM8 GB
GPUNVIDIA GeForce GTX 960 (4 GB) / AMD Radeon R9 380 (4 GB)
DirectXVersion 12
Storage40 GB available space (HDD supported, SSD recommended)
SoundDirectX-compatible sound card

These specs target 1080p resolution at 30 FPS with graphics settings on Low. The game is designed to run on hardware that's roughly 8-10 years old, making it accessible to most PC gamers. If your system meets or exceeds these specs, you can play the game — just don't expect maxed-out visuals.

Note for laptop users: The minimum GPU requirement is for desktop-class cards. A laptop GTX 1650 or equivalent mobile GPU with 4 GB VRAM should handle minimum settings. Integrated graphics (Intel UHD, Iris Xe, AMD Vega) are not supported and will not run the game at playable frame rates.

Recommended System Requirements (1080p / 60 FPS / High Settings)

OSWindows 11 64-bit
CPUIntel Core i7-8700K / AMD Ryzen 5 3600
RAM16 GB
GPUNVIDIA GeForce RTX 2060 (6 GB) / AMD Radeon RX 5600 XT (6 GB)
DirectXVersion 12
Storage40 GB SSD (NVMe recommended for faster load times)
SoundDirectX-compatible sound card

These specs deliver a smooth 1080p / 60 FPS experience on High settings. For 4K / 60 FPS on Ultra settings, you'll want at least an RTX 3070 or RX 6800 XT with 8+ GB VRAM. The game supports uncapped frame rates, so high-refresh-rate monitors (144 Hz, 240 Hz) are fully utilized on capable hardware.

Install Size & Storage Details

The base install size is 40 GB, with additional space needed for save files (approximately 500 MB for a full save with all collectibles) and future DLC content. The Deluxe Edition's DLC packs will add an estimated 5-10 GB over time. Plan for 50 GB total if you own the Deluxe Edition.

While the minimum specs list HDD support, we strongly recommend an SSD. Load times between the Batcave and Gotham's open world average 8-12 seconds on SSD versus 35-45 seconds on HDD. With the amount of zone transitions in the game, an SSD saves you hours of cumulative loading over a full playthrough.

Steam Deck Compatibility

LEGO Batman: Legacy of the Dark Knight is Steam Deck Verified at launch. TT Games has confirmed full Steam Deck support with the following performance profile:

  • Native resolution (800p): Medium-High settings, 40-60 FPS
  • Docked (1080p): Medium settings, 30-45 FPS
  • Battery life: Approximately 2.5-3 hours at 40 FPS cap

The game supports full controller input on Steam Deck with no additional configuration needed. Cloud saves via Steam Cloud are fully supported for seamless switching between desktop and Deck.

Performance Optimization Tips

Settings That Impact Performance Most

  1. Shadow Quality: Biggest FPS hit. Dropping from Ultra to Medium gains 15-20% FPS with minimal visual difference in a LEGO game.
  2. Anti-Aliasing: TAA (Temporal Anti-Aliasing) looks great and runs efficiently. Avoid MSAA 8x — it's overkill for this art style and costs significant performance.
  3. Draw Distance: Affects open-world Gotham more than linear missions. High is fine for most systems; Ultra only if you have headroom.
  4. Ambient Occlusion: Moderate FPS impact. SSAO is sufficient; HBAO+ adds little visual benefit in the game's brick-based aesthetic.
  5. Texture Quality: Set based on VRAM. 4 GB VRAM = Medium, 6 GB = High, 8+ GB = Ultra. Exceeding your VRAM causes stuttering.
Quick Optimization: If you're below 60 FPS, drop Shadows to Medium and set Anti-Aliasing to TAA first. These two settings alone often recover 20-30 FPS with barely any visual downgrade.

Ultrawide Monitor Support

The game fully supports 21:9 and 32:9 ultrawide resolutions natively — no ini tweaks or mods needed. Cutscenes are rendered in 16:9 with pillarboxing, but all gameplay and menus scale correctly to ultrawide aspect ratios.

Known Issues & Fixes

  • Stuttering on first launch: The game compiles shaders on first boot. This takes 2-3 minutes and causes temporary stuttering. Subsequent launches are smooth. Fix: Wait for shader compilation to complete before playing.
  • Controller not detected: If using a PlayStation controller on PC, enable PlayStation Controller Support in Steam's controller settings. Xbox controllers work natively.
  • Audio crackling: Rare issue on some Realtek audio chipsets. Fix: Set audio output to 16-bit 48000 Hz in Windows Sound Control Panel.
