LEGO Batman Legacy Won't Launch on PC? Here's Every Fix That Actually Works

Published: July 4, 2026 · 6 min read

Six weeks after LEGO Batman: Legacy of the Dark Knight launched on May 22, 2026, a small but persistent slice of the PC player base still can't get the game to open at all. On Reddit's r/legogaming, one player summed up the frustration a month into ownership: "So I purchased the Deluxe Edition a month ago... why can't I open up the game still?" It's not a widespread issue — most PC players have had a smooth launch — but for the players it does hit, the game either crashes to desktop on startup, hangs on a black screen, or never gets past the Steam/Epic loading spinner.

If that's you, here's a ranked troubleshooting checklist, based on the fixes the community and outlets covering the game have confirmed actually work, starting with the one most likely to solve it.

1. Verify the Game Files First

Corrupted or incomplete game files are the single most common cause of a game refusing to launch, and it's the fastest thing to rule out. On Steam, right-click LEGO Batman: Legacy of the Dark Knight in your Library, open Properties → Installed Files, and select Verify integrity of game files. The client will re-check every file against Steam's records and re-download anything that doesn't match. Epic Games Store users have the same option under the game's Manage menu — look for Verify. This single step resolves a large share of launch failures because it catches the download errors and disk corruption that cause everything downstream.

2. Reset the Config File

If verification doesn't fix it, the next confirmed step is clearing out a corrupted local configuration file. The game stores its config data at C:\Users\[your username]\AppData\Local\Dinner\Saved (Dinner is the internal codename TT Games uses for the project's Unreal Engine build folder). Rename the Config folder inside that path to Config.old — don't delete it outright in case you need to roll back — and relaunch the game. This forces the engine to regenerate a fresh configuration from scratch, which clears out settings corruption that a verify-files pass won't catch.

3. Force a DirectX 11 or 12 Launch Option

Some launch failures trace back to how the game negotiates with your GPU driver on startup. You can sidestep this by forcing a specific graphics API through a Steam launch option: open Properties → General on the game in your Steam Library, and enter -dx11 in the launch options box. If the game still won't start, try -dx12 instead. This won't fix every case, but it's a quick, reversible test that's worth running before you touch any Windows system settings.

4. Check Two Windows Security Settings

Two Windows 11 security features have been flagged as launch blockers for this game specifically, and both live inside the Windows Security app under App & browser control.

Force Randomization for Images

This setting is meant to harden memory allocation against exploits, but it can interfere with how the game's executable loads on startup. In Windows Security, go to App & browser control and set Force randomization for images (mandatory ASLR) to Off if it's been manually enabled.

Smart App Control

Smart App Control blocks apps and drivers it doesn't recognize as trusted, and a newly released game can occasionally get caught by that net before its reputation is established. Under the same App & browser control tab, open Smart App Control settings and switch it to Off. Note that this is a one-way setting on some Windows 11 builds — turning it back on later may require a clean install — so only disable it if the earlier fixes haven't worked.

5. Clear the Steam Download Cache

Steam keeps a local cache of connection and download data that can occasionally get corrupted without affecting other games. Clearing it is a low-risk step: open the Steam client, go to Settings → Downloads → Clear Cache (the exact path is documented on Steam's official support page), and Steam will restart and prompt you to log back in. This won't touch your installed game files, only cached network data.

6. Broader Fixes If Nothing Else Works

If the game still refuses to launch after the steps above, move on to general PC-level checks that apply to most stubborn launch failures, not just this game:

  • If your PC has both integrated and dedicated graphics, manually set the dedicated GPU as the game's renderer in your graphics driver's settings rather than relying on auto-detection.
  • Install the latest graphics driver directly from NVIDIA, AMD, or Intel rather than relying on Windows Update's bundled version.
  • Run Windows Update and install any pending Windows 11 updates — some launch failures are tied to OS-level API mismatches that get patched over time.
  • As a last resort, fully uninstall the game, delete the leftover C:\Users\[username]\AppData\Local\Warner Bros. Interactive Entertainment\LEGO Batman - Legacy of the Dark Knight folder, and reinstall from scratch.
Before you reinstall: Check our PC System Requirements guide first. A handful of launch failures reported in the community turned out to be minimum-spec hardware that could install the game but couldn't actually initialize it — reinstalling won't fix a hardware shortfall.

When to Contact Support Directly

If you've worked through every step above and the game still won't open, the next move is a direct support ticket through the official LEGO Games support portal, run by WB Games. At that point you're likely dealing with a system-specific conflict — antivirus software, a corrupted Windows install, or a driver issue — that's easier to diagnose with a support agent looking at your specific crash logs than through general troubleshooting steps.

For everything else once you're actually in the game, our Beginner's Guide and PC Requirements breakdown are the two most useful starting points, and our Post-Launch Patch Tracker covers every stability patch TT Games has shipped since release in case a future update addresses these launch issues directly.
