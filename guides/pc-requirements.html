LEGO Batman: Legacy of the Dark Knight launched on PC (Steam and Epic Games Store) on May 22, 2026. The game requires Windows 11 and a 50 GB SSD across all tiers — HDDs are not supported. It fully supports DLSS 3 (NVIDIA), FSR 3 (AMD), and XeSS (Intel) including frame generation on compatible hardware.

Upscaling Support — DLSS 3, FSR 3, XeSS

The game supports all three major upscaling and frame generation technologies. The minimum spec assumes upscaling is active — running at native resolution on a GTX 960 is not a supported configuration.

FSR 3 is available on all GPU brands (NVIDIA, AMD, Intel), making it the most universally accessible option. DLSS 3 Frame Generation requires an RTX 40-series GPU. XeSS frame generation requires an Intel Arc GPU.

Steam Deck Performance

The game runs on Steam Deck at 720p with FSR or XeSS in Performance mode, targeting a locked 30 fps on low-to-medium settings. The Xbox Series S performance comparison (720p/30fps) suggests comparable results. Battery life is approximately 1.5–2 hours during active gameplay. The game is not officially Steam Deck Verified as of launch — check the Steam page for updated certification status.

PC Performance Tips

  • Enable upscaling at launch — the game's settings menu defaults to native, but enabling DLSS/FSR Quality mode provides better image quality and significantly better frame rates than native on equivalent hardware.
  • 24 GB RAM for 4K — the additional RAM requirement at 4K is due to asset streaming for Gotham's open world. 16 GB causes noticeable texture pop-in at 4K.
  • Volumetric lighting setting — this is the single biggest GPU cost in the game. Dropping it from Ultra to High recovers 10–15% frame rate with minimal visual difference.
  • SSD speed matters — a NVMe SSD significantly reduces open-world load streaming compared to SATA SSD. SATA is supported but NVMe is recommended.
  • RTX 3060 Ti at 1440p — review benchmarks confirmed the RTX 3060 Ti handles 1440p/60fps comfortably with DLSS Quality enabled, making it the value sweet spot for this title.

Post-Launch Patch Notes (PC)

Since the May 22 launch, TT Games has released the following PC-specific updates:

  • Day-one patch (May 22): Fixed a crash affecting RTX 20-series GPUs when entering Central Island's business district with ray tracing enabled. FSR 3 frame pacing issue corrected. VRAM usage reduced by approximately 400 MB at Ultra texture settings.

Keep the game updated via Steam or Epic Games Store to receive performance patches automatically. The game has an active patch cadence — check the Steam news page for the latest updates.

Frequently Asked Questions

Is Windows 10 supported?

No. Windows 11 (64-bit) is required. This is not a soft recommendation — Windows 10 is not on the official supported OS list. The UE5 DirectStorage implementation the game uses for open-world asset streaming requires Windows 11's storage stack.

Does the game work on Linux / Steam Deck?

The game runs on Steam Deck at 720p/30fps via Proton (low-to-medium settings, FSR Performance mode). Full Linux desktop support via Proton is functional but not officially supported. The game is not yet Steam Deck Verified as of May 29, 2026 — check the Steam listing for updated certification status.

Can I play on a laptop?

Yes, with caveats. A laptop GPU equivalent to the GTX 1660 Ti mobile or better handles the minimum spec. Laptop GPUs with 6–8 GB VRAM perform best. Ensure your laptop is plugged in during play — the game's asset streaming is demanding enough that battery-saver throttling causes noticeable stuttering even on powerful hardware.

Why does the game require an SSD? Can I use an HDD?

The game uses DirectStorage for open-world asset streaming in Gotham City — continuously loading and unloading building assets, pedestrians, and environmental detail as you traverse. A mechanical HDD cannot sustain the read speeds required, resulting in visible texture pop-in and hitching. An SSD (SATA or NVMe) is a hard requirement, not a suggestion.

How large is the download?

The base game install is approximately 50 GB on all platforms. The Deluxe Edition DLC (Mayhem Collection) adds approximately 4–6 GB when it releases in September 2026. A 50 GB allocation on an SSD covers the base game with room for patches.
