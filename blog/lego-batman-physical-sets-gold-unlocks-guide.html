All 4 LEGO Batman Sets With In-Game Gold Unlocks in Legacy of the Dark Knight

Published: July 1, 2026 · 5 min read

Four physical LEGO Batman sets released alongside LEGO Batman: Legacy of the Dark Knight each carry a bonus that only exists in the game itself: a QR code on the building instructions that unlocks an exclusive golden character or vehicle skin. This is separate from the account-linked free suits (Golden Age Batsuit via WB Games account, Steel Dark Knight and Black Lantern suits via HBO Max) and separate from the Dark Knight Returns Batsuit preorder bonus. These four unlocks require actually owning the physical brick set.

The Four Sets and What Each One Unlocks

LEGO's official redemption page confirms exactly four sets carry a Legacy of the Dark Knight tie-in code, each unlocking a different golden item:

Set Price In-Game Unlock
Batman Logo (76330)$79.99Golden Batman character
Batman v Superman Batmobile (76331)$29.99Golden Batmobile from Batman v Superman: Dawn of Justice
The Batman Batmobile (76332)$29.99Golden Batmobile from The Batman (2022)
Batman & Robin Batmobile (76333)$29.99Golden Batmobile from Batman & Robin

The Batman Logo set (76330) is the only one that unlocks a playable character skin — the Golden Batman — rather than a vehicle. It also has no in-game prerequisite: register the set and the code works immediately.

The Catch: Gold Batmobiles Need In-Game Progress First

The three Batmobile sets work differently from the Batman Logo set. Per LEGO's own redemption page, to actually see the gold paint job you must first unlock the standard black version of that specific Batmobile in-game by completing the relevant side quests — only then does redeeming the code swap it for the golden variant. Buying the set alone doesn't hand you the gold vehicle; it hands you the code, and the code only pays off once you've put in the driving-and-side-quest work to unlock the base model first.

Practically, that means players chasing the Batman v Superman, The Batman, or Batman & Robin gold Batmobiles should prioritize Gotham's open-world side content before expecting the code to do anything visible.

How to Redeem the Code (3 Steps)

  1. Create or sign in to a LEGO.com account — this is separate from your WB Games or HBO Max account used for the other free-suit unlocks.
  2. Scan the QR code printed on the cover of the set's building instructions using a smartphone camera.
  3. Register the set to receive a redemption code, then enter that code in LEGO Batman: Legacy of the Dark Knight through the game's in-game redemption system.
Timing: Codes can be registered to a LEGO profile at any time — before or after the set is built — but can only actually be redeemed in-game from launch day onward. LEGO states codes are guaranteed valid until March 1, 2029, and each account can redeem only one code per set.

Is It Worth Buying a Set Just for the Game Content?

None of these four unlocks affect gameplay stats — they're cosmetic swaps, the same category as the game's 23 Red Bricks (also purely cosmetic in Legacy). If you were already planning to pick up a Batman LEGO set for the shelf, the in-game code is a genuine bonus at no extra cost. If you're buying purely for the pixels, the $29.99 Batmobile sets are the cheaper entry point, though remember the gold paint won't show until you've cleared the matching Batmobile's side quests. The $79.99 Batman Logo set is the only route to the Golden Batman character and the only one of the four with no in-game prerequisite attached.
